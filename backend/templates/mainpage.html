This web application is a prototype front end interface for utilizing convolutional neural networks (deep learning) for automatic generation of DICOM-compliant structure set file.
Current support is for head and neck CT studies. The workflow will generate contours for the following OAR:

  • Brain
  • Brainstem
  • Spinal Cord
  • Parotid Glands (left and right)
  • Cochlea (left and right)
  • Submandibular Glands (left and right)
  • Larynx
  • Brachial Plexus
